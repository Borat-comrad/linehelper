# Streamlit UI Demo

## Аудит текущего интерфейса

Главный файл Streamlit-интерфейса: `linehelper/ui/streamlit_app.py`.

До обновления интерфейс был функциональным, но минимальным:

- верхняя часть состояла из короткого заголовка `LineHelper` и подписи MVP;
- основная зона работала как чат поверх локального RAG;
- источники выводились обычным списком под ответом;
- диагностика была всегда доступна в expander, но не отделялась от пользовательского сценария;
- боковая панель, режимы работы, быстрые сценарии и демонстрационные бейджи отсутствовали;
- загрузки документов, отдельного блока КП и отдельного 1С-блока в текущем Streamlit UI нет.

Существующие блоки:

- чат пользователя и ассистента;
- источники ответа;
- диагностика retrieval/LLM;
- настройки поиска были только в backend-константах и env, без UI;
- формирование КП и 1С-контекст пока представлены как сценарии демонстрации, без отдельной backend-интеграции.

## Что изменилось

- добавлена шапка `LineHelper — локальный помощник Serviceline`;
- добавлен подзаголовок про корпоративную базу знаний, КП и локальную память;
- добавлены бейджи `semantic`, `episodic`, `1C context`, `local only`;
- добавлена боковая панель с режимом работы, параметрами поиска и статусом локального контура;
- добавлены быстрые запросы для демонстрации;
- источники вынесены в `st.expander` и показываются карточками;
- карточки источников показывают document title, namespace, score, source, page/section и фрагмент;
- semantic и episodic источники различаются мягким визуальным акцентом;
- технические детали скрыты за чекбоксом `Показывать технические детали`;
- CSS вынесен отдельно, без внешних ассетов, CDN и frontend-фреймворков.

## Затронутые файлы

- `linehelper/ui/streamlit_app.py` - новая структура Streamlit-экрана.
- `linehelper/ui/components.py` - форматирование бейджей и карточек источников.
- `linehelper/ui/styles.py` - локальный CSS для корпоративного вида.
- `tests/test_ui_components.py` - минимальные тесты форматирования источников.
- `docs/STREAMLIT_UI_DEMO.md` - это описание.

## Запуск

```powershell

```
