# Глоссарий

| Термин | Простое объяснение |
| --- | --- |
| LineHelper | Локальный AI-агент Serviceline, сейчас на этапе MemoryStore MVP |
| RAG | Подход, где модель сначала ищет нужные источники, а потом отвечает с опорой на них |
| semantic memory | Память для стабильных знаний: инструкции, регламенты, справки |
| episodic memory | Память для подтвержденного практического опыта, который может устаревать |
| chunk | Небольшой фрагмент текста, который отдельно хранится и ищется |
| embedding | Числовое представление текста для смыслового поиска |
| BM25 | Алгоритм ранжирования полнотекстового поиска |
| FTS5 | Модуль полнотекстового поиска SQLite |
| rerank | Повторная сортировка найденных кандидатов по качеству и контексту |
| TTL | Срок жизни записи |
| orchestrator | Слой, который решает, какие инструменты вызвать для задачи пользователя |
| 1С API | Интерфейс для получения актуальных операционных данных из 1С |
| КП | Коммерческое предложение |
| source citation | Ссылка на источник ответа: документ, страницу, раздел, запись памяти |
| smoke test | Быстрая проверка, что основной сценарий не сломан |
| tech debt | Технический долг: известная проблема или упрощение, которое нужно исправить |
| ADR | Architecture Decision Record, запись об архитектурном решении |
