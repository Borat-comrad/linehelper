# Интеграция с 1С

В коде репозитория нет модулей интеграции с 1С, endpoints, клиентов API или скриптов обмена. Все ниже, кроме принципа "1С не является памятью", нужно уточнить у владельца проекта.

## Роль 1С

Гипотеза: 1С должна быть источником актуальных операционных данных для ответов и КП:

- цены;
- остатки;
- номенклатура;
- статусы заказов;
- данные клиентов;
- условия поставки;
- история сделок, если доступна.

## Почему 1С не память

Память LineHelper должна хранить знания и подтвержденный опыт. 1С хранит актуальное состояние бизнеса. Если скопировать оперативные данные в память, они могут устареть и привести к ошибочным ответам.
