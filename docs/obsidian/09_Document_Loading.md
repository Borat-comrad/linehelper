# Загрузка документов

## Риски

- PDF может иметь плохой текстовый слой.
- DOCX-таблицы могут потерять смысл при простом извлечении текста.
- Документы могут содержать персональные или коммерчески чувствительные данные.
- Повторная загрузка без дедупликации засорит память.
- Нужна политика, какие документы можно индексировать.
