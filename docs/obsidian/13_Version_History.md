- `фактически найдено`: подтверждено git log или файлами проекта.
- `план`: желаемый будущий этап, не выпущен.
- `черновик`: структура версии описана предварительно.
- `гипотеза`: предположение, требующее проверки.
- `не подтверждено Git`: нельзя выдавать за релиз.
